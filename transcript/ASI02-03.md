--------------------------------------------------
TERMINOLOGY RECAP
--------------------------------------------------

Before we move on, let's reinforce the key concepts from this scenario.

Tool Output Poisoning.

Compromised or manipulated data returned from a legitimate enterprise tool.

Tool Output Trust Boundary.

The controls that determine whether tool output is sufficiently trusted for downstream decisions.

Analysis-to-Execution Collapse.

Analysis intended for human review is incorrectly treated as execution authority.

Zero-Trust Tooling.

Every tool output must be independently validated before it is allowed to influence execution.

Agentic Decision Contamination.

The agent's planning process is influenced by compromised intermediate outputs.

Unauthorized Trade Execution.

Real capital moves because analysis output bypasses validation and approval controls.

And that's the most important lesson from this scenario.

The trading engine was never compromised.

(Pause.)

The agent trusted analysis that should never have become authority.
