Entry point

Indirect prompt injection

Attacker content arrives as normal data and is interpreted like an instruction.

Boundary

External untrusted input boundary

Where external material enters the agent's working context.

Failure point

Trust boundary failure

The agent stops separating trusted system guidance from untrusted external content.

Core mechanism

Agentic reasoning loop

The read-decide-act cycle where the agent interprets context and takes action.

Category outcome

Agent goal hijack

The business objective is silently replaced or redirected.

Decision impact

Control flow manipulation

The next action changes because the decision path has already been corrupted.

Key insight

The injection happens before the reasoning loop finishes. By the time the agent decides, the goal is already wrong.
